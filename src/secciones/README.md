# Paquete de secciones

Este paquete reúne el código Python por funcionalidad:

- clasificacion.py: parsing y OCR de la clasificación.
- resultados.py: parsing de resultados de partido.
- proximo_partido.py: construcción de datos de próximo partido.
- resultados_jornada.py: lectura y escritura del JSON de resultados de liga.
- ultimo_resultado.py: localización del último asset de resultado.
