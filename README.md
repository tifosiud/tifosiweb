# Tifosi Web

Proyecto organizado por responsabilidades:

- src/bot: punto de entrada del bot de Telegram.
- src/processing: lógica de generación de imágenes, OCR y utilidades de procesamiento.
- src/secciones: módulos por funcionalidad (clasificación, resultados, próximo partido, etc.).
- web: frontend estático (HTML, CSS y JavaScript).
- data: datos JSON del proyecto.
- imagenes: imágenes generadas.
- tests: pruebas unitarias.
